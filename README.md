A fully offline, privacy-first, zero-equipment bodyweight training app built with Flutter.

**Precision Fitness & Flow** guides you through a flexible Hybrid Weekly workout plan (Precision Flow & Military Task) consisting of targeted strength intervals and intense HIIT (High-Intensity Interval Training) flows. It features a stunning, distraction-free Chronos-Timer and relies on absolutely zero external dependencies, user accounts, or internet connections after the initial download.

## ✨ Key Features
- **Hybrid Weekly Plan**: Choose between Precision Flow (technique) and Military Task (performance) daily based on your form.
- **Advanced Chronos Timer**: An intuitive, visually engaging timer guiding you exactly when to push and when to rest.
- **Dynamic Training Experience**: Voice coach announcements (TTS), haptic vibration feedback, and wakelock integration for a seamless hands-free workout flow.
- **Zero Privacy Concerns**: No data collection, no forms, no tracking, and no external APIs. All your workout history stays strictly on your local device.
- **12 Core Exercises**: Push-Ups, Squats, Lunges, Mountain Climbers, Burpees, Glute Bridges, Plank, Wall Sit, Pull-Ups, Sprint/High Knees, Long Run, and Stretching—each with minimalist visual form guides.
- **Adaptive Execution Guides**: Specific joint-safety warnings (especially for plyometric exercises) to prevent injuries.

## 🛠 Tech Stack
- **Framework**: [Flutter](https://flutter.dev/) (Dart) (Cross-Platform compilation to Web, Android, iOS)
- **Local Database**: [Hive](https://docs.hivedb.dev/) (Fast NoSQL document tracking)
- **State Management**: [Riverpod](https://riverpod.dev/) (Reactive, testable, null-safe)
- **Routing**: [GoRouter](https://pub.dev/packages/go_router)
- **Offline Audio**: [Audioplayers](https://pub.dev/packages/audioplayers)

## 🚀 Installation & Build

Because Flutter is beautifully cross-platform, you can build this natively for Android, iOS, or Web.

### 1. Requirements
- Flutter SDK `^3.16.0` or higher
- Dart SDK `^3.0.0` or higher

### 2. Getting Started
Clone this repository and grab the dependencies:
```bash
git clone https://github.com/USERNAME/precision-fitness-flow.git
cd precision-fitness-flow
flutter pub get
```

### 3. Generate Hive Adapters
The app uses Hive for local data storage. You need to build the generated adapters:
```bash
flutter pub run build_runner build --delete-conflicting-outputs
```

### 4. Run the App
To run it on your local browser (Chrome) for quick feedback:
```bash
flutter run -d chrome
```

To compile a highly optimized web-release folder (found in `build/web`):
```bash
flutter build web --release
```

## 🔒 Privacy Policy
Precision Fitness & Flow collects **no personal data**. No user account is required to use this application. All training history generated by you is stored purely locally on your device's internal storage via Hive. Once the application is downloaded/loaded, an internet connection is no longer required. 

> **💡 Note on the Web Version (GitHub Pages):**
> Even when using the app in your browser, your progress and statistics are securely saved in the browser's local database (IndexedDB). As long as you don't clear your browser data or use Private/Incognito mode, your training history will seamlessly persist across multiple sessions!
